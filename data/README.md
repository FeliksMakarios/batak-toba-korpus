# data/

Folder ini berisi berkas korpus dalam format CSV.

Tempatkan dua berkas berikut di sini:

- `btb_multigenre.csv` - korpus multi-genre (sastra, media kontemporer, pendidikan, historis/tradisional, religius).
- `btb_bible.csv` - korpus paralel Alkitab Batak Toba dan Bahasa Indonesia.

Kedua berkas menggunakan skema kolom yang sama. Definisi setiap kolom dan taksonomi genre dijelaskan pada [../metadata/skema-kolom.md](../metadata/skema-kolom.md).

---

This folder contains the corpus files in CSV format.

Place the following two files here:

- `btb_multigenre.csv` - the multi-genre corpus.
- `btb_bible.csv` - the parallel Batak Toba and Indonesian Bible corpus.

Both files share the same column schema. Each column and the genre taxonomy are described in [../metadata/skema-kolom.md](../metadata/skema-kolom.md).
